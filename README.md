# btnav

Mobile app to control relay over BLE to stay on a fixed course

Prerequisites: https://reactnative.dev/docs/environment-setup

```
cd BTNav
yarn install
yarn start # in terminal 1
yarn android # in terminal 2
yarn android:build
yarn android:install
```
